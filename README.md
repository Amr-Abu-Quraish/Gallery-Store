# GalleryStore



## 📝 Project Overview
This is a responsive e-commerce product gallery built with Angular and TailwindCSS.  
It fetches products from Fake Store API and allows searching, sorting, and viewing product details.

## 🧱 Tech Stack Used
- Angular 19
- Tailwind CSS
- Fake Store API
